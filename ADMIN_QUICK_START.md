# Admin Quick Start Guide

## 🚀 Get Your Exam Portal Running in 5 Minutes

### Step 1: Login as Admin (30 seconds)

1. Open the Lab Examination Portal
2. Click **"Admin Panel →"** at the bottom of login page
3. Enter credentials:
   - **Username**: `sscet`
   - **Password**: `adminsscet@2026`
4. Click **"Login as Admin"**

### Step 2: Register Students (2 minutes)

1. Click **"Manage Students"** button (top right)
2. Click **"Add Student"**
3. For each student, enter:
   - **Student Name**: Full name (e.g., "John Doe")
   - **Register Number**: ID (e.g., "2024CS001")
4. Click **"Add Student"**
5. Repeat for all students

**Quick Tip**: Keep names and register numbers simple and easy to remember. Students must enter these EXACTLY as registered.

### Step 3: Create Your First Question (2 minutes)

1. Go back to Admin Panel (click "Back to Admin")
2. Click **"Manage Questions"**
3. Click **"Add New Question"**
4. Fill in the form:

**Question Details:**
```
Title: Write a program to sort an array
Description: Write a program to sort the array [64, 34, 25, 12, 22, 11, 90] in ascending order and print the result.
Programming Language: JavaScript (or choose: Python, Java, C, C++)
Difficulty: Medium
Expected Output: [11, 12, 22, 25, 34, 64, 90]
```

**Viva Questions (Scroll down):**

Add 10 viva questions. Here are 3 sample questions to get started:

**Question 1:**
```
Question: What is the time complexity of Bubble Sort?
Option 1: O(n)
Option 2: O(n log n)
Option 3: O(n²)
Option 4: O(log n)
Correct Answer: Option 3
```

**Question 2:**
```
Question: Is Bubble Sort a stable sorting algorithm?
Option 1: Yes
Option 2: No
Option 3: Sometimes
Option 4: Depends on implementation
Correct Answer: Option 1
```

**Question 3:**
```
Question: What is the space complexity of Bubble Sort?
Option 1: O(1)
Option 2: O(n)
Option 3: O(log n)
Option 4: O(n²)
Correct Answer: Option 1
```

_(Continue adding 7 more questions...)_

5. Click **"Save Question"**

### Step 4: Ready to Go! ✅

Your system is now ready. Students can:
1. Login with their registered credentials
2. Get a random question
3. Take the exam
4. Submit and download results

### Step 5: After Exams - Evaluate Observations

1. Go to **Admin Panel**
2. View all student submissions in the table
3. For each student:
   - Click the **pencil icon** in "Actions" column
   - Enter **Observation Marks** (0-10)
   - Click **"Save"**
4. Total marks will update automatically
